
issue:

1. 编译时，出现 undefined reference 问题

原因：未找到函数的实现，
        1) 链接时缺失了相关目标文件（.o）
        2) 链接时缺少相关的库文件（.a/.so）
        3) 链接的库文件中又使用了另一个库文件
        4) 多个库文件链接顺序问题（根据库之间的依赖顺序）
        5) 在c++代码中链接c语言的库
        解决：添加一个extern "C"的声明，并包含c语言库相关的头文件
               extern "c"
               {
               #include "xxx.h" 
               }

查看库文件中实现那些函数
objdump
